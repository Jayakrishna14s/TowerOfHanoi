Tower of Hanoi

Game rules

  • Move one disk at a time.
  • Larger disks cannot be placed on smaller disks.
  • Disks are stacked in ascending order on rods.
  • Only the top disk of a rod can be moved.
  • Solve recursively.

How to play?

  • Game is played moving discs from one rod to other rod
  • Game consists of 3 rods
  • Select a number to choose number of discs to play with
  • Solve recursively following the rules

Rules
  • Each move involves selection of two rods
  • Disc moves from first selected rod to second selected rod
  • Selection of rods are done by mouse clicks on respective rods
  • For n number of discs, minimum number of moves to move all discs from one rod to another is 2^(Number of discs) - 1
  • Larger disks cannot be placed on smaller disks.
  • Only the top disk of a rod can be moved.
